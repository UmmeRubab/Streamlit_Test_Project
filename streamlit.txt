What is requirements.txt?

A text file that lists all Python packages your project needs

Example:

streamlit==1.28.0
pandas==2.1.0
numpy==1.26.0


When someone clones your repo, they can install all dependencies with:

pip install -r requirements.txt


Do not upload your virtual environment, only this file

How to Create requirements.txt

Make sure your virtual environment is activated:

(streamlitwork) E:\AI and Data Science Course - AnalytixCamp\Streamlit_Projects>


Run:

pip freeze > requirements.txt


pip freeze lists all installed packages in your environment

> requirements.txt writes them to the file

Add it to Git:

git add requirements.txt
git commit -m "Add requirements file"
git push
